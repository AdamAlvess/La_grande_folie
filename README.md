# La Grande Folie

Voici notre jeu de ferme réalisé en cours de POO.

Le principe est de créer une ferme qui va tourner automatiquement, qui va évoluer afin de faire face à des intempérie et aux conditions de travail difficiles.

Installation UV

Commande UV : 
- uv run python -m ferme.minimal_logiciel (lancer le client)
- uv run python "commande dans le read me chronobios" (lancer le serveur)
- uv run python launch.py  pour tout lancer en meme temps

On utilise GitHub action. A chaque push de la branche Tests, tout les test sont éffectués. A la fin, un mail de rapport des tests est envoyé.
